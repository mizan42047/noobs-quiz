Noobs Quiz plugin comes with quiz solution for your website. You can use our plugin Either with elementor or block.We added both block and elementor support.


== Description ==
Noobs Quiz plugin is help you to add quiz fuctionality in your website.It has lots of feature and some interesting is upcomming. 

=== Create Question From Dashboard ===
Just install and activate the plugin you will got a option in your dashboard menu just click there and add new you can add question.first write your question to the title box.the click plus icon and search question meta.you will find fully block supported meta box there.


=== Add Question Options ===
1. Add Question correct answer 
2. Add mark to the question
3. Add options for your question as much you can
